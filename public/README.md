# Static Assets Directory

Place the following images in this directory:

## Required Images

1. **hero-bg.png**
   - Hero banner background image
   - Used in the main hero section at the top of the page
   - Can be any abstract tech/gradient background

2. **arrow-path.png**
   - Diagonal arrow graphic
   - Used in the "Path to Copilot" section showing progression
   - Should be a diagonal arrow pointing up-right

## Optional Logos

3. **microsoft-logo.svg**
   - Microsoft logo for partner section
   - Optional - will display as text if missing

4. **sentinel-logo.svg**
   - Microsoft Sentinel logo
   - Optional - will display as text if missing

## Note

The page will function without these images, but some visual elements will be missing or display placeholder content. You can:
- Add actual images matching these names
- Update the Next.js Image components in `app/page.tsx` to point to different paths
- Remove the Image components if not needed
